New items are inserted into itemsJson.json.

{
	"items":
	[
		{
			"ID": "testShirt",
			"itemName": "Test Shirt",
			"slotTag": "chest",
			"itemSprite": "wardrobeTempResources/shirtTEST",
			"itemDescription": "This is a shirt made for testing",
			"coversBottomPiece": false
		}
	]
}

Breakdown of example item:
	"ID" is a unique identifer for the code and internal systems, NOT the name but should be similiar.
	"itemName" is the displayed name of the item.
	"slotTag" this is the slot the item goes into; it can be "chest", "bottom", "shoe", or "jacket",.
	"itemSprite" this is the path to the image, this path does not start from the "Assets" folder it starts from "Resources" folder. If you want your image to appear you MUST have the item in the Resources folder.
	"itemDescription" this is the displayed description of the item.
	"coversBottomPiece" will hide the equiped bottom piece, as of writing I have not implemented this function yet.
